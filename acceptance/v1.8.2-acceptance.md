## 本地签名与权限说明

- 本轮使用的是本地生成的 OpenHarmony simulator signing material，不是 AppGallery Connect 生产签名材料
- 本地 release profile 的 `restricted-permissions` 为空；模拟器安装时会拒绝缺少 profile grant 的 restricted permission
- 为了记录 release HAP 的最低安装/启动链路，ccnubox 本地生成物未把 `ohos.permission.NOTIFICATION_CONTROLLER` 与 `ohos.permission.READ_IMAGEVIDEO` 带入最终安装用 HAP
- 因此这条证据不覆盖 image picker、notification、camera 等 capability 的 release runtime 行为
